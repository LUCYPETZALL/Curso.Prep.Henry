ARRAYS

Los ARRAYS son matrices. Las matrices son contenedores de almacenamiento para coleccionar datos

Para estructurar un ARRAY debemos:

1) Construir una matriz. Usamos la funcion 'const'

2) Declarar una variable y establecerla entre [];

3) agregar elementos (str, numeros o booleanos) al contenedor (se usa la ',' como separador)

Ej: nombresFamilia = ['Juan','Marie','Eze','Ito','Martin']

Podemos acceder a un elemento de la matriz usando un indice [i] asi:

console.log(nombresFamilia)[3]); //'Eze'

El primer elemento de una matriz tiene asignado el valor 0

Para conocer el numero de elementos de un array aplicamos ".length" :

Ej: console.log(nombresFamilia.length); //5

Para acceder al ultimo elemento de una matriz aplicamos .length-1

Ej: console.log(nombresFamilia [nombresFamilia.length-1]; //'Martin'

Se puede cambiar cualquier elemento de la matriz de la siguiente manera:

nombresFamilia [0] = 'Pirulo'

console.log(nombresFamilia); //['Pirulo','Marie','Eze','Ito','Martin']

Para agregar un elemento al final de una matriz usamos ".push"

Ej: nombresFamilia.push ('Santiago') ubicara ese nombre despues de 'Martin y aumentara la longitud de la matriz

Para eliminar el ultimo elemento de la matriz y disminuir su longitud en 1 usamos ".pop":

nombresFamilia.pop(); borra el nombre 'Santiago' que habiamos agregado con .push

".unshift" agrega un elemento al principio de la matriz, y, ".shift" elimina el primer elemento de la matriz
